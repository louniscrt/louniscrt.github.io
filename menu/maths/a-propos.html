Architecture du site

Application de révision Maths S2 pour BUT R&T.

Ce site est une application web statique en HTML, Tailwind CSS et JavaScript. Il est organisé pour être facilement transformé plus tard en application PHP/MySQL avec connexion, scores persistants et suivi utilisateur.

Fiches de cours

Les notions sont organisées par chapitre avec définitions, théorèmes, méthodes, exemples et erreurs fréquentes.

TD interactifs

Les exercices sont présentés avec des indices progressifs et une correction découpée étape par étape.

Progression

Les meilleurs scores de quiz et les validations manuelles sont sauvegardés localement dans le navigateur.

Évolution possible

Préparation PHP/MySQL

Phase suivante
01

Connexion utilisateur

Ajouter une table users avec email, pseudo et mot de passe hashé.

02

Scores persistants

Remplacer le localStorage par une table quiz_attempts.

03

Progression réelle

Stocker les fiches vues, les TD terminés et le meilleur score par chapitre.

04

Administration

Créer une interface pour ajouter facilement des cours, exercices et quiz.

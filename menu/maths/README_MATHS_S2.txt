Application Maths S2 - BUT R&T

Emplacement conseillé : /menu/maths/
Page d'accueil : /menu/maths/index.html

Fichiers principaux :
- index.html : dashboard
- cours.html / cours.js : fiches de cours avec MathJax
- td.html / td.js : TD interactifs avec indices et corrections
- quiz.html / quiz.js : quiz de validation avec score localStorage
- progression.html / progression.js : suivi local
- a-propos.html : documentation du projet
- data.js : données pédagogiques temporairement en dur
- style.css : design cyber/tech

Le projet utilise Tailwind CDN, FontAwesome CDN et MathJax CDN.
Les scores sont stockés en localStorage et pourront ensuite être migrés vers PHP/MySQL.
